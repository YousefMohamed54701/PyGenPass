## 🖥️ What is PyGenPass?

PyGenPass is a desktop app built with Python. It generates secure, customizable passwords. You can set how long the password is, what characters it uses, and more. - Generate passwords of any length  
- Choose to include letters, numbers, and symbols  
- Save your favorite settings for quick access  
- Copy passwords to clipboard with one click  
- Clear and simple user interface with purple colors  
- Runs on Windows without extra software

## 🔒 How to Use PyGenPass

Here is how you generate a safe password.

1. Open the PyGenPass app.

2. Choose the length of your password by moving the slider or entering a number. Common lengths are 12 or more characters for better security.

3. Select which characters to include:  
   - Letters (both uppercase and lowercase)  
   - Numbers  
   - Symbols (like !, @, #, etc.)  

4. Click the **Generate** button. You will see a new password appear in the box below.

5. To use the password, click the **Copy** button. This copies the password to your clipboard.

6. Paste the password into the site or app where you want to use it.

7. If you want another password, click **Generate** again.
